Project Overview

This project implements an end-to-end Image Caption Generation system using a ResNet-50 convolutional encoder and a Transformer-based decoder. The model is trained on the Flickr8k dataset and supports both greedy decoding and beam search for high-quality caption generation.

Problem Statement

Image captioning combines computer vision and natural language processing, requiring the model to understand visual content and generate syntactically and semantically coherent sentences. This is a central problem in multimodal AI research, and the architecture implemented here follows the modern encoder–decoder paradigm widely used in research publications.

Key Techniques Used

  • ResNet-50 feature extraction
  • Transformer decoder with multi-head attention
  • Custom vocabulary and text preprocessing
  • Teacher forcing during training
  • Greedy and beam search decoding
  • GPU acceleration using Apple Silicon (MPS)
