Training Observations

Training loss consistently decreases across epochs, demonstrating stable optimization. Validation loss remains lower due to teacher forcing. BLEU scores show expected fluctuations between epochs when using greedy decoding.

Key Insights

  • The Transformer decoder significantly improves linguistic quality compared to LSTM baselines.
  • Beam search decoding produces more coherent and contextually rich sentences.
  • ResNet-50 provides strong feature representations even without fine-tuning.
  • Apple Silicon (MPS) provides a substantial speed-up over CPU-only training.

Limitations

  • The encoder produces a single global feature vector, limiting spatial attention.
  • Small dataset size (Flickr8k) restricts linguistic diversity.
  • BLEU evaluation currently uses a single reference per caption.

Future Extensions

  • Use spatial feature maps for attention-based visual grounding.
  • Train and evaluate on MS COCO for improved linguistic coverage.
  • Implement length normalization and coverage penalties in beam search.
  • Fine-tune the visual encoder for domain adaptation.
  • Integrate a Vision Transformer (ViT) encoder.
